# UI Design Variations

Here are different versions of the modern engineer-style portfolio design:

## Variation 1: Current Design (Terminal/Code Style)
- Grid backgrounds
- Terminal window frames
- Code-style elements
- Teal/cyan color scheme
- Split hero layout

## Variation 2: Minimalist Terminal
- Cleaner, more minimal
- Less grid patterns
- More white space
- Simpler borders
- Focus on content

## Variation 3: Dark Terminal/Matrix Style
- Darker background
- More terminal/console aesthetic
- Green/cyan accents (like terminal)
- More code-like typography
- Command-line inspired elements

## Variation 4: Sidebar Navigation
- Fixed left sidebar navigation
- Main content area
- More structured layout
- Code-style sidebar

## Variation 5: Card-Heavy Grid
- More card-based design
- Stronger grid system
- Better visual separation
- More structured sections

## Variation 6: Color Accent Variations
- Purple/blue instead of teal/cyan
- Orange/red accents
- Blue/indigo theme
- Green/emerald theme
