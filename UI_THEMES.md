# UI Theme Variations

Here are different UI styles you can try. Each has a unique aesthetic:

## 1. Space Theme (Current)
- Dark space background with stars
- Teal/cyan color scheme
- Grid patterns
- Revolving planet
- Modern engineer aesthetic

## 2. Minimal Theme
- Clean, simple design
- Lots of white space
- Subtle colors
- Thin borders
- Focus on content

## 3. Retro Theme
- 80s/90s computer aesthetic
- Pixelated elements
- Retro colors (purple, pink, cyan)
- CRT screen effect
- Nostalgic feel

## 4. Dark Terminal Theme
- Terminal/console aesthetic
- Green/amber on black
- Monospace fonts everywhere
- Command-line feel
- Matrix-like elements

## 5. Colorful Theme
- Bright, vibrant colors
- Gradient backgrounds
- Playful design
- High contrast
- Energetic feel

## 6. Brutalist Theme
- Bold, raw design
- Strong borders
- High contrast
- Geometric shapes
- No-nonsense aesthetic

Let me know which one you want to try, or I can implement a theme switcher!
